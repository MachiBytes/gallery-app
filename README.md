# TUP Manila - Introduction to Cloud Computing Workshop

Welcome to the **Introduction to AWS Workshop** repository! This project demonstrates core AWS services through practical implementation.

## Workshop Overview

This workshop showcases the following AWS services:
- **VPC** - Virtual Private Cloud for network isolation
- **EC2** - Elastic Compute Cloud for application hosting
- **RDS** - Relational Database Service for data storage
- **S3** - Simple Storage Service for file storage and static website hosting

## Project Structure

```
TUP_WORKSHOP/
├── README.md                 # This file
├── gallery-app/             # Full-stack image gallery application
│   ├── README.md            # Application setup and deployment guide
│   ├── server.js            # Express.js backend
│   ├── package.json         # Node.js dependencies
│   ├── .env.example         # Environment variables template
│   └── public/              # Frontend files
└── static-website/          # S3 static website
    ├── index.html           # Landing page
    ├── styles.css           # Styling
    └── assets/              # Images and other assets
```

## Getting Started

1. **Gallery Application**: Navigate to `gallery-app/` and follow the README instructions
2. **Static Website**: Deploy the `static-website/` folder to S3 for static hosting

## Workshop Learning Objectives

- Set up AWS VPC with proper security groups
- Deploy applications on EC2 instances
- Configure RDS for database operations
- Use S3 for file storage and static website hosting
- Implement environment-based configuration
- Deploy production applications using PM2
